# Game Backlog Tracker

This is a school project where users can manage their game backlog. Users can add games, edit game details, and delete games from the list. Each game includes a title, platform, and status.

## Features

- Add new games
- Edit game title, platform, and status
- Delete games
- Responsive layout

## Built with

- HTML
- CSS
- JavaScript

## How to run

Open in the browser
